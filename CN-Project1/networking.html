1) DNS (Domain Name System)

DNS translates a human-friendly domain name into IP addresses that computers use to route traffic.

Evidence

  • Command: nslookup your-domain
  • Screenshot: show the domain queried + returned IP addresses

2) IP Addressing (IPv4 vs IPv6)

IPv4 uses 32-bit addresses (A records). IPv6 uses 128-bit addresses (AAAA records).

Evidence

  • IPv4 (A record): nslookup -type=A your-domain
  • IPv6 (AAAA record): nslookup -type=AAAA your-domain
  • Screenshots: outputs for both commands

3) Protocols (HTTP vs HTTPS)

HTTP sends traffic in plaintext. HTTPS encrypts traffic using TLS, protecting confidentiality and integrity.

Evidence

  • HTTP headers: curl.exe -I http://your-domain
  • HTTPS headers: curl.exe -I https://your-domain
  • Screenshot: show status codes (200/301/302) and key headers

4) Security Feature Implemented

Security feature: HTTPS (encrypted connections). GitHub Pages supports HTTPS. We verify HTTPS and (ideally) an HTTP → HTTPS redirect using header output.

Evidence

  • Screenshot: curl.exe -I https://your-domain output
  • Screenshot: curl.exe -I http://your-domain output (shows redirect if present)
